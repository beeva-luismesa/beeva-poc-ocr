AWS EMR - Processing Layer - EMRFS
EMRFS is an implementation of HDFS which allows EMR clusters
to store data on Amazon S3
Features:
Server-side and client-side encryption
Allows working with encrypted data on EMR
Consistent view for EMRFS
Consistency checking for list and read-after-write (for new
put requests) for objects in Amazon S3. Enables that any
data stored in a step is available from consequent steps.
EMRFS metadata in Amazon DynamoDB
